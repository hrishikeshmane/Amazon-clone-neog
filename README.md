# Amazon Clone

https://amzn-clone-neog.netlify.app/

## To run locally:

```
    git clone https://github.com/hrishikeshmane/Amazon-clone-neog.git

    And open the index.html file in browser
```

## Features:

- Home page
- Product feed
- Filter sidebar
- Cart
- Wishlist
- Image
- Input
- Mobile responive
